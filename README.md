# Todo HTTP API

Простой HTTP-сервис для управления списком задач (TODO), написанный на Go.
Хранилище данных — in-memory (без использования внешних БД).

Проект реализован в рамках тестового задания и использует только стандартную библиотеку Go.

## Возможности

- Создание задачи
- Получение списка всех задач
- Получение задачи по идентификатору
- Обновление задачи
- Удаление задачи
- Валидация входных данных
- Корректные HTTP-статусы и JSON-ответы
- Unit-тесты для основных кейсов

## Запуск локально

Убедитесь, что у вас установлен Go (версия 1.22 или выше).

```bash
go run ./cmd/app
```

После запуска сервер будет доступен по адресу:
```
http://localhost:8080
```

## Запуск с помощью Docker

### Сборка образа
```bash
docker build -t todo-api .
```

### Запуск контейнера
```bash
docker run -p 8080:8080 todo-api
```

После этого API будет доступно по адресу:
```
http://localhost:8080
```

## API

Описание HTTP API и контрактов находится в OpenAPI-спецификации:

```
docs/openapi.yaml
```

## Тесты

Для запуска всех unit-тестов выполните:

```bash
go test ./...
```

## Примечания

- Проект не использует сторонние библиотеки
- Контекст (`context.Context`) прокидывается через слои
- Архитектура разделена на domain / repository / service / transport
